# @vxna/glsl-loader
Bundle your shaders without comments

## Usage

**webpack.config.js**

```js
module.exports = {
  module: {
    rules: [
      {
        test: /\.(frag|vert|glsl)$/,
        loader: '@vxna/glsl-loader'
      }
    ]
  }
}
```
